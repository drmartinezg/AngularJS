# DepositAccountsApplication

Deposit Accounts Management with Angular and Gulp.

## Dependencies
- Angular.js
- Angular-Route
- Angular-Resource
- Bootstrap
- Fontawesome

## Usage
### Install global
```
$ npm install -g stylus
$ npm install -g gulp
$ npm install -g bower
```

### Install local
```
$ npm install
$ bower install
```
### Development mode
```
$ gulp
```
### Go-to-production
```
$ gulp build
$ gulp server-dist
```
